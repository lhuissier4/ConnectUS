# ConnectUS
Projet Open Innov de l'EPSI

# Frontend
Il s'agit d'un projet NextJS créé via la commande `npx create-next-app@latest frontend`

# Backend
Il s'agit d'un projet NestJS initialisé avec `nest new backend`

# Dévelopement
Clonez le projet
```bash
git clone https://github.com/lhuissier4/ConnectUS.git
cd ConnectUS
```

## Initialiser les projets
```bash
git submodule update --init --recursive
cd frontend 
git checkout main
npm i 
cd ../backend
git checkout main
npm i
```
## Démarrer l'application
Dans le dossier backend, executez :
```bash
npm start
```
Rendez-vous dans le dossier frontend dans un autre terminal et executez : 
```bash
npm start
```
